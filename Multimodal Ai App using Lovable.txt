Build a modern production-ready AI Screenshot Explainer web application using Next.js 15, TypeScript, Tailwind CSS, and Gemini 2.5 Flash API.

Project Name:
AI Screenshot Explainer

Goal:
Users can upload any screenshot, image, error screenshot, dashboard screenshot, code screenshot, website screenshot, or application screenshot. The AI should analyze the image and provide a detailed explanation of what is visible in the screenshot.

Core Features:

1. Screenshot Upload
- Drag and drop upload area
- Support PNG, JPG, JPEG, WEBP
- Maximum file size 10 MB
- Preview uploaded image before analysis

2. Gemini Vision Analysis
- Use Gemini 2.5 Flash API
- Analyze uploaded screenshot
- Extract visible text
- Understand image context
- Explain what is happening in the screenshot
- Detect possible errors if visible
- Provide suggestions and recommendations

3. Analysis Sections
Display AI response in professional cards:

A. Screenshot Summary
- What is shown in the screenshot

B. Extracted Text
- OCR-style extracted text

C. Context Understanding
- Explain screen purpose and content

D. Issues Detected
- Potential errors or warnings

E. Recommendations
- Suggested next actions

4. Loading Experience
- Professional skeleton loaders
- Progress indicators
- Smooth animations
- Streaming-like experience

5. History Section
- Store last 10 analyses in local storage
- Allow reopening previous analyses

6. Copy Features
- Copy analysis
- Copy extracted text
